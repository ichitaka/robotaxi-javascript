Spielanleitung

Sie haben 30 € zur Seite gelegt und wollen nun diese geschickt investieren. Sie entscheiden sich in autonome Taxis zu investieren. Diese können ohne Ihr Zutun Fahrgäste einsammeln und so Geld für Sie verdienen. Leider funktionieren diese Maschinen aber nicht immer perfekt. Deshalb müssen Sie auch für alle Schäden die aufkommen haften.

In diesem Spiel können Sie mehrmals in verschiedene autonome Fahrzeuge investieren. Jede Fahrt kostet Sie dabei 1 €. Ihr Ziel ist es möglichst viel Geld mit der richtigen Investition in insgesamt 30 Runden zu sammeln. Sie erhalten eine Auszahlung in Höhe von 10% Ihres finalen Kontostandes nach Abschluss dieses Experiments.

{% if condition == 1 %} Während jeder Runde schauen Sie dem Taxi zu, wie es Geld für Sie verdient. Während jeder Runde werden Sie über Ihre Webcam aufgezeichnet. {% elif condition == 2 %} Während jeder Runde schauen Sie dem Taxi zu, wie es Geld für Sie verdient. Dabei sollen Sie die Leistung des Taxis mit Ihren Reaktionen bewerten. Reagieren Sie auf gute und negative Ereignisse mit Ihrer Mimik. Das Taxi wird versuchen daraus zu lernen. Während jeder Runde werden Sie über Ihre Webcam aufgezeichnet. {% else %} Während jeder Runde schauen Sie dem Taxi zu, wie es Geld für Sie verdient und Ihre Reaktionen zu der Leistung des Taxis werden für Forschungszwecke aufgezeichnet. {% endif %}

In der ersten Runde können Sie das Taxi selbst steuern, damit Sie das Spiel verstehen.

Sammeln Sie einen Fahrgast ein, verdienen Sie damit 6 €.
Dieses Auto sollten Sie vermeiden! Sollte es zur Kollision kommen, müssen Sie 5 € Strafe zahlen.
Wenn Sie diese Straßensperre beschädigen, müssen Sie eine Strafe von 1 € bezahlen.
Die Bäume markieren die Grenzen des Spielfeldes. Wenn Sie die Grenze erreichen, wird das Auto automatisch in eine zufällige Richtung wechseln.
